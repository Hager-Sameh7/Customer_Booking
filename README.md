# ✈️ Flight Operation Performance Dashboard (Excel)

## 📌 Overview

This project is an interactive **Flight Operation Performance Dashboard** built using **Microsoft Excel**.
It analyzes flight booking data to uncover insights about customer behavior, sales performance, and operational efficiency.

The dashboard is designed to help stakeholders quickly monitor KPIs and make data-driven decisions.

---

## 🎯 Objectives

* Track and evaluate booking performance
* Analyze customer booking behavior across different dimensions
* Identify factors affecting conversion and abandonment rates
* Highlight opportunities for improving sales channels and upselling

---

## 📊 Key Metrics (KPIs)

* **Conversion Rate:** 15%
* **Average Passengers per Booking:** 2
* **Extra Baggage Rate:** 67%
* **Average Lead Time:** 85

---

## 📈 Dashboard Insights

* A high **abandonment rate (85%)** suggests potential friction in the booking process
* **Internet** is the dominant sales channel compared to Mobile
* Peak booking activity occurs during **6–11 AM**
* **RoundTrip bookings** tend to have shorter purchase lead times
* Strong adoption of **extra baggage** indicates upselling success

---

## 🧩 Features

* Interactive filters using slicers:

  * Flight Day

  * Sales Channel

* Sales channel performance analysis

## 📂 Dataset

The dataset contains booking-related information, including:

* Booking status (completed / abandoned)
* Trip type (One Way, Round Trip, Circle Trip)
* Sales channel (Internet, Mobile)
* Booking time and day
* Passenger details
* Ancillary services (e.g., extra baggage)

## 🚀 How to Use

1. Download the Excel file from this repository

3. Use the slicers to filter data dynamically
4. Explore KPIs and charts for insights
